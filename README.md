# Task Management Web Application

A simple **Full-Stack Task Management Web Application** built to manage daily tasks.  
This project demonstrates basic **frontend and backend integration**, RESTful APIs, and data persistence.  

---

## **Assessment Overview**

This project was developed as part of a **full-stack development assessment** for internship-level evaluation.  
The focus is on:

- Creating, viewing, updating, and deleting tasks
- Marking tasks as **Completed** or **Pending**
- Implementing **RESTful APIs** for frontend-backend communication
- Basic form validation and error handling

---

## **Features**

- Create a task with **title** and **description**
- View a list of all tasks
- Edit task details
- Delete a task
- Toggle task status: **Completed / Pending**
- Responsive and user-friendly interface

---

## **Technical Stack**

| Layer        | Technology Used                 |

| Backend      | Node.js, Express.js             |
